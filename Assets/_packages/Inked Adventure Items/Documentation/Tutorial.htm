Tutorial

Requirements

If you want to make modifications to the existing icons or draw your own, you can, of course, use your preferred image editing software and reproduce the style, but if you want to draw in an identical fashion, I would recommend giving Krita a shot. It's a free and amazing painting software I can't recommend enough. For digital drawing, I'm using a Wacom graphics tablet, but a mouse will do if necessary. For actual image editing in terms of cropping, positioning and post-processing the final icons, I use the free software GIMP, but for this asset pack, I generate the icons directly from my Krita file with the help of a little tool.

This tutorial is limited to explaining how the icons were created and how the given Krita source file Showcase.kra is structured, so I won't cover how Krita works; check out their tutorials for that purpose.

Colors

The color palette covers:

#ffffff White (for special cases only)
#efefef Light
#cfcfcf Medium
#b8b8b8 Dark
#454545 Lining

I will refer to these colors as Light, Medium, Dark and Lining. I'm not using actual black and white for the linings and filling, because it looks kind of harsh. If you need them, you can increase the contrast in the sprite sheets with an image editing program.

Style Variants

The icons come in four variants: Black and White (bw), Grayscale (grayscale), Light (light), White (white) and Lining (lining). All of them use the color Lining for the linework.

The variant Lining is fully transparent within and outside of the linework: it's the best starting point when you want to make your own fillings. The variants White is filled with White within the linework as a kind of special case. I would rather recommend Light, which is filled with the Light color instead. The Grayscale variant is my favorite: it's filled with Medium, while shadows have the color Dark and highlights are drawn with Light. The final variant Black and White has a Light filling and shadows in the Lining color with additional linework in Light on top.

Brushes

Now to the most important part, the brushes. In Krita, the brush used for the linings is called ink-1 precision. It can be found in the Ink category. I've edited the brush settings of this brush so Size and Softness don't respond to Speed as well as so there's no change in Opacity depending on the pen pressure (if you draw with a graphics tablet). If you don't change that, fast movements with the mouse or pen will lead to gaps in the drawing, which is something we don't want if we do linework.


Ink-1 Precision

Ink-2 Fineliner

For the sketches, I draw with ink-1 precision in brush size 3.0, opacity 10% and color Lining. The actual linework, including outlines and inner details, are done on a separate layer on top, in size 3.0 and opacity 100% with the same brush type and color.

The filling is done on a separate layer below the linework's layer. You can hide the sketching layer after you finished the linework. The filling colors depend on the style variant you want to reproduce. They were explained in the previous section. For filling out larger sections, I'll recommend to use ink-2 fineliner with the same changes to the brush settings as explained above. I've actually created an additional ink-1 precision and ink-2 fineliner variant where the size grows on pressure, which makes filling out smaller and larger parts even more comfortable. But that's not really necessary.

Drawing

The source file Showcase.kra comes with a layer group ICON BACKGROUNDS in which you'll find a layer corner light which contains icon backgrounds consisting of four little marks in the corner of each icon. It's the most fitting one to draw your own icons in. You could copy some of those and paste them below the other icons to have a row of empty icon backgrounds to draw in.

Zoom in while drawing and, if required, rotate the canvas to be more accurate when drawing in certain angles. The brush tool has a stabilizer you can toggle on to make longer lines more smoothly. Draw by moving the wrist, not the arm. If you do linework, put Undo and Toggle Eraser on some shortcut keys (one of my left fingers is usually placed on one of them).

Style Guidelines

The following guidelines refer to how this specific set of icons was drawn.

  • Be abstract enough, so the items can be identified from their silhouette.
  • Don't be too detailed. Use little cracks and irregularities (see the Stone item) sparingly.
  • Detail lines try to stand free from any other lines (check the little lines on the tool item handles like the Pickaxe), except if there isn't enough space (see Carrot) or if it's rough wood (compare Bark with Wood and Root).
  • Don't put too many detail lines in close space. One accentuation is usually enough.
  • Detail lines often consist of a line to indicate the 3-dimensional shape of the object (like if it's curvy), then followed by a little dot or shorter line. This is a recurring style element. Check out the Beet, the Mushroom, the Pickaxe and the Glove as examples.
  • Items which are held in a hand have a cloth wrapping at the location where they are held (see Sword or Bow). The wrapping always has the same color.
  • When coloring, Medium is the main color in the Grayscale style variant and Light in the Black and White variant. These colors should take the most space for most of the items.
  • Lines do not collapse to form thicker lines.
  • The color Lining should only be used as a filling color in the Grayscale variant, if it reaches inside the item (see Boot or Skull) or in special cases where it's kind of expected (see Bomb).
  • The light comes usually from the left or top-left, except if the item points with its biggest surface into a different direction (see Backpack) or if the item is shown in a symmetric fashion (see Armor).
  • A single round dot as a highlight on the biggest surface is preferred (see Liquid or Apple).
  • Grass patches and feathers have a fluffyness to them (see Arrow or Turf).
  • Similar items look similar (see Crate and Barrel or Stick and Branch).
  • All items with a handle (check the tool items) have a similar looking but not identical handle, with different cloth wrapping and detail lining.
  • Use a single line without an inner part only if it's a thread or line which couldn't be replaced with, let's say, a leather strap. See Bow or Fishing Rod, which would look confusing if the line had a thickness. Compare that to the Ball of Wool or the Amulet, where you can get away with a thick line.
  • Try to fill out the whole icon space, even if the item is very small in comparison (compare the sizes of the Key with the Armor as an example).
  • Flames consist of a main flame and little secondary flames. The main flame has one inner part in a darker color. Compare Campfire with the Lantern.

Filter Layer Trickery

As I had to draw the filling in various styles, I didn't want to re-draw the basic filling color which fills out the whole inner part of the linework over and over again. If you have a look at the ICONS→ICON FILLING PLAIN layer group, you'll notice a layer called white. It contains the actual filling of the icons in White. On top, there is a layer medium. It's a filter layer Adjust→HSV/HSL Adjustment, which, when visible, modifies the color of the white filling into a Medium color for the Grayscale style variant. Correspondingly, the filter layers light 1 and light 2, if both visible at the same time, change the white filling into the Light color for the Black and White and Light style variants (the medium has to be hidden and vice versa). I had to run the white filling through the two filter layers light 1 and light 2, because I couldn't exactly hit the Light color with one HSV/HSL adjustment alone.

Source File Structure

The topmost layer groups are:
ICONS Contains all the icon linework, sketches and fillings.
ADDITIONAL Contains the item names and the white banner behind the names.
ICON BACKGROUNDS Contains various item backgrounds you can switch through to see the resulting icons with different item backgrounds.
BACKGROUND The showcase background, including the logo and decorational elements. Hide this whole group if you want to export icons.

Within the ICONS layer group, you'll find the following groups:

ICON LININGS The heart of the source file. The linework is split into a lining layer with the important lines and a details layer for additional decoration. The layer bw white overlining should be made visible only if you want to display the Black and White style variant, as it contains lighter detail linework which only makes sense on the dark shades of the filling layer in the ICON FILLING BW layer group.
ICON FILLING BW Contains the Lining-colored filling. The lighter detail linework can be found in the ICON LININGS layer group above.
ICON FILLING GRAYSCALE Contains the icon fillings for the Grayscale variant. The layer shade ink holds shading in the Lining color for special cases, like the eyeholes of the skull icon or the bomb icon's body.
ICON FILLING PLAIN Contains very basic filling within the icons' linework. The previous section explains the effect layers medium, light 1 and light 2.
ICON SKETCHES Contains rough little sketches I drew the lineworks on. Hidden by default.

Just Draw

You don't have to use the complicated layer group structure I've used. I had to, because I've got a little tool which cuts out the icons from this showcase file and generates all the single icon files and sprite sheets in the different variants. If I were you, I would make a bunch of new layers on top for the sketches, the linings and the single filling colors and draw over the existing icons (if you want to make modifications) or copy a bunch of icon backgrounds below the last row of icons and draw new icons there.

Trick to Move Icons

To move icons as a whole, across all layer groups and layers within, use the Rectangular Selection tool and select the icon. Then, instead of using the Move tool, use the Transformation tool instead, as the movement done with this one can be applied on a layer group. The Move tool only works on single layers, sadly.
